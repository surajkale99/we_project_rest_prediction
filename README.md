# 🍽️ Restaurant Reviews Prediction System using NLP & Scikit-learn

## 📌 Project Overview
This project is a **Restaurant Reviews Prediction System** built using **Natural Language Processing (NLP)** and **Scikit-learn**.  
The system analyzes customer reviews and predicts whether the review is **Positive** or **Negative**.

It uses text preprocessing techniques such as:
- Tokenization
- Stopword Removal
- Stemming
- Vectorization

Machine Learning algorithms are then applied to classify the sentiment of reviews.

---

# 🚀 Features
- Text preprocessing using NLP
- Convert text into numerical vectors
- Train ML classification models
- Predict review sentiment
- Accuracy evaluation
- Easy-to-understand implementation
